Данная версия программы может быть использована с пультами без внешней Flash-памяти AT45DB641.
Загрузка крупных объектов данных (описатели меню) производится непосредственно в памяти процессора STM32.
Начальный адрес размещения в памяти STM задается параметром FLASH_OPIS_START_ADDR, который находится в microprog.h.
Для возврата работы с внешней памятью AT45DB642 добавить  #define BLOCK_STM_MEMORY
В этом случае полностью сохраняется работоспособность программы пульта.

Внесено изменение в процесс загрузки уставок в память ПЧ. При передаче в ПЧ значительно снижается количество пакетов в пульт,
в результате чего есть вероятность критического снижения питания и сброса пульта. Поэтому для более стабильной работы на время 
передачи данных в ПЧ выключается подсветка индикатора. Это позволило существенно стабилизировать передачу данных на любом качестве
соответствия пульта и ложемента.

Добавлена возможность тестирования программы под JTAG без загрузок прошивки из ПЧ. Для этого надо добавить #define SUBSTITUTE_BOOTLOADER
в этом случае подменяются параметры Bootlaoder в PULT_BOOT FRAM, тогда контрольная проверка CheckSum будет проходить нормально.
Без этого объявления, при отладке каждый запуск PULT_BOOT будет пытаться загрузить программу из ПЧ.

Добавлена информация в начало программы о версии ПО. (Искать  #ifdef VERSION_SECTION в main.c)
Эта возможность выполнена при помощи добавления вспомогательной секции памяти, которая размещена по адресу  0x8010000-0x80100FF.
Ранее основная программа размещалась начиная с адреса 0x8010000. После добавления секции основная программа сдвинулась на 0x100 
и теперь находится по адресу 0x8010100 (внесены соотв. изменения в файла Линкера icf и передвинута таблица прерываний)
Информация о версии программы формируется на основании ранее использованных объявлений в файле VersionPo.h:
#define GRP_VERSION_PO_PU24E        
#define SUB_VERSION_PO_PU24E        
#define MRK_VERSION_PO_PU24E        
Версия программы объявлена c адреса - 0x8010008: младшие два байта - версия программы пульта, старшие два байта - сигнатура программы.
static const u32 PrgVersion @ 0x8010008 = (u32)(GRP_VERSION_PO_PU24E*1000+SUB_VERSION_PO_PU24E) + (u32)((u32)(MRK_VERSION_PO_PU24E) << 16);

Первые 8 байт повторяют из основной программы значение Main Stack Pointer и Reset_Handler. Перемещение данных величин ЗАПРЕЩЕНО!
В противном случае программа не запускается из под PULT_BOOT.

Также в заголовок добавлен текстовый фрагмент "VERSION HEADER", который отображает наличие секции с заголовком версии, что можно использовать
для идентификации в конфигураторе AT24.
